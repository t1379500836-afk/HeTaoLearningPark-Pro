# 打字练习文档

> 四种练习系统概述 + 代码位置指引

---

## 一、四种练习系统

| 组件 | 练习内容 | 数据来源 | 适用场景 |
|------|---------|----------|----------|
| **TypingPractice** | 单词模式 + 代码模式（英文） | lesson-data.js + typing-templates-pool.js | Python代码语法练习 |
| **ChineseTypingPractice** | 古诗 + 成语（中文） | chinese-typing-pool.js | 中文输入练习 |
| **ShortcutPractice** | 电脑快捷键 | 内置数据 | 快捷键操作练习 |
| **代码模板池** | 难度分级代码模板 | typing-templates-pool.js | 随机代码练习 |

---

## 二、功能特性

### 虚拟键盘交互

- **布局**: QWERTY 5行标准键盘
- **按键状态**:
  - 按下：橙色高亮 (#ff9f00)
  - 正确：绿色渐变
  - 错误：红色渐变
- **自动恢复**: 300ms
- **响应式**: 支持桌面/平板/手机多端适配

### 进度追踪

| 统计项 | 说明 |
|--------|------|
| 字母/分钟 | `(正确字符数 / 用时秒数) × 60` |
| 准确率 | `(正确字符数 / 总字符数) × 100%` |
| 用时 | `分:秒` 格式 |

### 排行榜系统

- 保存前5名成绩
- 支持跨练习持久化
- 成绩对比（与上次练习）
- 儿童友好版提示词

### 代码模板池 (typing-templates-pool.js)

**文件**: `src/data/courses/PY2/typing-templates-pool.js`

| 难度 | 模板数量 | 说明 |
|------|---------|------|
| `easy` | 131条 | 单行代码、基础语法 |
| `medium` | 120条 | 多行代码、常见操作 |
| `hard` | 97条 | 复杂逻辑、嵌套结构 |

**使用方式**:

```javascript
import { getRandomTemplates } from '@/data/courses/PY2/typing-templates-pool.js'

// 随机抽取8个基础难度模板
const templates = getRandomTemplates(8, 'easy')

// 随机抽取5个全部难度模板
const templates = getRandomTemplates(5, 'all')
```

**模板示例**:

| 难度 | 示例 |
|------|------|
| easy | `for i in s:`, `s.split(" ")`, `max(scores)` |
| medium | `for char in s:\n    print(char)`, `items = text.split(",")` |
| hard | `for i, char in enumerate(s):`, `[n * 2 for n in range(1, 6) if n % 2 == 0]` |

## 四、Props 完整列表

### TypingPractice

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `mode` | `'word' \| 'code'` | `'word'` | 练习模式 |
| `customWords` | Array | `[]` | 自定义单词列表 |
| `customTemplates` | Array | `[]` | 自定义代码模板 |
| `showLevelSelector` | Boolean | `false` | 显示关卡选择器 |
| `embedded` | Boolean | `false` | 嵌入式模式（隐藏排行榜） |
| `wordCount` | Number | `8` | 每轮练习数量 |
| `scoreHistory` | Array | `[]` | 外部管理的排行榜数据 |

### Events

| Event | 说明 |
|-------|------|
| `complete` | 练习完成时触发 |
| `restart` | 点击"返回"或"重新开始"时触发 |
| `update:scoreHistory` | 排行榜数据更新时触发 |
