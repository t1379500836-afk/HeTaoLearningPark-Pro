## 二、路由配置

### 路由表

| 路由 | 组件 | 说明 |
|------|------|------|
| `/` | HomeView | 首页（个性化显示） |
| `/levels` | CourseLevelsView | 阶段选择（PY1/PY2/PY3） |
| `/levels/:stage` | StageView | 单元选择（L1-L18） |
| `/levels/:stage/:unit` | UnitView | 课时选择 |
| `/lesson/:stage/:unit/:lesson` | LessonView | 课时主页面 |
| `/practice` | PracticeView | 课后练习 |
| `/typing` | TypingView | 独立打字练习 |
| `/python` | PythonIDEView | 独立Python编辑器 |
| `/ycl` | YCLZoneView | YCL专区 |
| `/ycl/practice/:level` | YCLPracticeView | YCL练习选择 |
| `/ycl/exam/:level/:setId` | YCLExamView | YCL考试界面 |
| `/contact` | ContactView | 联系作者 |
| `/locked` | StageLocked | 阶段锁定提示 |

> **注意**：以上路由都支持 `p1/`、`py2/`、`python3/` 三个前缀版本

### 路由前缀权限

| 前缀 | 可访问阶段 | 适用学生 |
|------|-----------|----------|
| 无前缀 | PY1, PY2, PY3 | 管理员/完整访问 |
| `p1/` | PY1 | Python入门学生 |
| `py2/` | PY1, PY2 | Python进阶学生 |
| `python3/` | PY1, PY2, PY3 | Python高级学生 |

## 三、组件关系

### 页面组件复用关系

| 页面组件 | 复用的子组件 |
|----------|-------------|
| HomeView | HeroSection |
| TypingView | TypingPractice, ChineseTypingPractice, ShortcutPractice |
| PythonIDEView | CodeEditor |
| LessonView | TypingPractice + CodeEditor + FlashcardDisplay + KnowledgeCard + ExerciseCard |

### 课时页面结构

```
LessonView.vue
├── Breadcrumb（面包屑导航）
├── 单词卡 (FlashcardDisplay)
├── 知识点 (KnowledgeCard × N)
├── 打字练习 (TypingPractice)
└── 课后习题 (ExerciseCard × N)
```

## 四、数据文件结构

### lesson-data.js 结构

每个课次的独立数据文件：`src/data/courses/{阶段}/lessons/{课次}/lesson-data.js`

```javascript
export const lessonMeta = { id, title, subtitle, ... }    // 课次元数据
export const vocabData = [...]                            // 单词卡数据
export const knowledgePoints = [...]                      // 知识点（3个难度）
export const exercises = [...]                            // 习题数据
export const typingWords = { easy, medium, hard }        // 打字练习单词
export default { meta, vocab, knowledgePoints, exercises, typingWords }
```
