# 宝塔部署指南（IP:8080）

## 上传文件

从 `HeTao/` 文件夹上传到服务器 `/www/wwwroot/hetao/`（不含 `server/node_modules`）。

上传后目录结构：

```
/www/wwwroot/hetao/
├── dist/                    # 用户端
├── admin/dist/              # 管理端
├── server/                  # 后端（不含 node_modules）
├── ecosystem.config.cjs
└── deploy/
```

## 服务器命令

```bash
# 1. 安装 Node.js（宝塔终端执行）
source /www/server/nvm/nvm.sh
nvm install 18 && nvm use 18 && nvm alias default 18

# 2. 安装后端依赖
cd /www/wwwroot/hetao/server
npm config set registry https://registry.npmmirror.com/
npm install --omit=dev

# 3. 修改数据库配置
#    用宝塔文件管理器编辑 /www/wwwroot/hetao/server/.env
#    填入实际的 DB_PASSWORD

# 4. 启动后端
cd /www/wwwroot/hetao
pm2 start ecosystem.config.cjs
pm2 save

# 5. 验证（应看到"数据库初始化完成"和"后端服务已启动"）
pm2 logs --lines 10
```

## Nginx 配置

宝塔 → 网站 → 添加站点（域名随便写，根目录选 `/www/wwwroot/hetao`，PHP 选纯静态），然后替换配置文件：

```nginx
server
{
    listen 8080;
    server_name _;

    location / {
        root /www/wwwroot/hetao/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    location /admin {
        alias /www/wwwroot/hetao/admin/dist;
        index index.html;
        try_files $uri $uri/ /admin/index.html;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    access_log /www/wwwlogs/hetao.log;
    error_log /www/wwwlogs/hetao.error.log;
}
```

## 放行端口

宝塔 → 安全 → 放行 `8080`，**同时**云服务商安全组也要放行 `8080`。

## 访问地址

| 地址 | 页面 | 账号 |
|------|------|------|
| `http://IP:8080` | 用户端 | 无需登录 |
| `http://IP:8080/admin` | 管理后台 | admin / admin123 |

## 常用命令

```bash
pm2 list                    # 查看状态
pm2 restart hetao-api       # 重启后端
pm2 logs hetao-api --lines 20  # 查看日志
nginx -t                    # 检查 Nginx 配置
nginx -s reload             # 重载 Nginx
```
