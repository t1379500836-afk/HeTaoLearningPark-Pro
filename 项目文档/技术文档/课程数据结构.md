## 二、数据结构详解

### vocabData（单词卡）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `word` | String | ✅ | 单词 |
| `pronunciation` | String | ✅ | 音标（避免 `'`，用 `-` 替代） |
| `partOfSpeech` | String | ✅ | 词性（n./v./adj.等） |
| `meaning` | String | ✅ | 中文释义 |
| `level` | String | ✅ | 难度：easy/medium/hard |
| `example` | String | ✅ | 英文例句 |
| `exampleTranslation` | String | ✅ | 例句翻译 |

### knowledgePoints（知识点）

| 字段 | 说明 |
|------|------|
| `id` | 唯一标识（如 'kp-1'） |
| `title` | 知识点标题 |
| `emoji` | 图标 |
| `gradeLevel` | 目标年级（'1-2'/'3-4'/'5-6'） |
| `summary` | 一句话概括 |
| `easy/medium/hard` | 3个难度版本，结构相同 |

**难度版本结构**:

| 字段 | 说明 |
|------|------|
| `story` | 趣味故事导入 |
| `concept` | 核心概念讲解 |
| `syntax` | 语法格式（`\n` 表示换行） |
| `example.code` | 可执行的代码示例 |
| `example.output` | 代码运行输出 |
| `example.explanation` | 输出解释 |
| `practice` | 思考题数组 `[{question, answer}]` |

### exercises（习题）

| 字段 | 说明 |
|------|------|
| `id` | 唯一标识（如 'ex-1'） |
| `level` | 难度：easy/medium/hard |
| `levelLabel` | 显示标签（基础/进阶/挑战） |
| `levelEmoji` | 图标（🟢/🟡/🔴） |
| `type` | 题型（当前只支持 'multiple-choice'） |
| `mathConcept` | **数学概念（必填！）** |
| `question` | 题目内容（支持Markdown） |
| `options` | 选项数组（**不要加 "A. " 前缀**） |
| `answer` | 正确答案索引（0=第一个选项） |
| `explanation` | 解析（支持Markdown） |
| `hint` | 提示内容（可选） |

## 三、命名规范

### 课程标题 (title)

**规则**: 使用**故事化名称**，避免技术性描述

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| split命令疯狂原始人 | 疯狂原始人 |
| 查找思想智取核桃山 | 智取核桃山 |
| 列表排序忍无可忍 | 忍无可忍 |

### 课程副标题 (subtitle)

**格式**: `学会[知识点1]和[知识点2]`

| 课程 | subtitle |
|------|----------|
| L7-1 | 学会遍历和分割字符串 |
| L7-2 | 学会查找最大最小值和列表推导式 |
| L7-3 | 学会列表排序和求和 |

## 四、内容数量标准

| 数据类型 | 数量要求 | 说明 |
|---------|---------|------|
| 单词卡 (vocabData) | 4-6个 | OCR提取的单词必须包含，至少拓展1个 |
| 知识点 (knowledgePoints) | 2-4个 | 每个知识点需要3个难度版本 |
| 习题 (exercises) | **6道** | 2基础 + 2进阶 + 2挑战 |
| 打字单词 (typingWords) | 每难度4个 | easy/medium/hard 各4个 |

### 习题难度分布

| 难度 | 数量 | 数学概念示例 |
|------|------|-------------|
| 🟢 easy | 2道 | 数数、符号识别、找规律 |
| 🟡 medium | 2道 | 分组问题、位置索引、算式计算 |
| 🔴 hard | 2道 | 植树问题、周期问题、变量计算 |

### 知识点三梯度标准

| 难度 | 目标年级 | 内容特点 |
|------|----------|----------|
| 🟢 easy | 1-2年级 | 故事场景导入、概念简单、代码直白 |
| 🟡 medium | 3-4年级 | 实际应用场景、概念深入、代码变体 |
| 🔴 hard | 5-6年级 | 综合运用场景、算法思维、优化技巧 |

## 六、常见问题

### Q: 音标中有特殊字符导致报错？

使用 `-` 替代 `'`：

```javascript
// ❌ 错误
pronunciation: "[trə'vɜːs]"

// ✅ 正确
pronunciation: "[trə-vɜːs]"
```

### Q: 为什么选项不要加 "A. " 前缀？

ExerciseCard 组件会自动添加选项标签，手动添加会导致重复。

### Q: 如何处理代码输出中的换行？

使用 `\n` 表示换行：

```javascript
output: 'A\nB\nC'  // 会渲染为三行
```
